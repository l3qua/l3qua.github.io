Commands List

Slash Commands

Command | Description | Example

/help | Display a link to the list of available commands | /help

/membercount | Count how many members are in your server | /membercount

/clear [amount] | Delete some messages from a channel | /clear amount:66

/kick [user] [reason] | Kick a member in the server | /kick user:@tester01#0001 reason:test

/warn [user] [reason] | Warn a member | /kick user:@tester01#0001 reason:test

Command | Description | Example | (Requirements)
